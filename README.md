## About

아리카에서 다중 로봇 관제, 맵 편집, 운영형 웹 서비스를 개발하고 있습니다.  
복잡한 현장 요구사항을 화면 UX, API, 데이터 흐름, 운영 문서까지 이어 붙여 실제 운영자가 바로 사용할 수 있는 제품으로 만드는 역할을 맡고 있습니다.

## Current Work

- **ARC Fleet 다중 로봇 통합 관제 플랫폼**
  - ARC Fleet 운영 콘솔의 초기 구조와 핵심 기능 설계를 맡고, 프론트엔드와 백엔드를 직접 구현
  - React · TypeScript · NestJS 기반으로 프로젝트/로봇/그룹/모델 관리, 실시간 관제, 수동 제어, 맵 편집·보정, 명령 이력, 배포 상태 확인 기능 설계·개발
  - 삭제 가드, 미저장 이탈 방지, 대표 지도 변경 경고 등 운영 실수를 줄이기 위한 가드레일 반영
- **국립평창청소년수련원 로봇 운영·콘텐츠 통합 시스템**
  - 안내·이송·순찰 로봇 운영을 위한 관리자 웹, 콘텐츠 관리 웹, 운영 서버, 배포·운영 환경 개발
  - Next.js 14 기반 관제 화면, Express 기반 웹서버, 통계 API, 코드 업로드·초기화, 콘텐츠 업로드, TTS·이메일 연동 기능 개발
  - 프로젝트에 포함된 이송 로봇 `뉴비(Neubie)`의 Nav2 bringup, keepout/narrow mode, docking/undock, elevator board/alight 운용 및 안정화
  - 3D 라이다를 2D scan으로 변환하는 파이프라인과 TF/scan 주기를 조정해 주행 안정성과 CPU 사용량 문제 대응
  - 이중화 서버 이슈, nginx·인증서 설정, 운영 절차를 문서화해 현장 대응 기반 정리
- **RobotApp Web RViz / SLAM 운영 도구**
  - FastAPI + WebSocket + rclpy 기반 ROS2 직접 연동 게이트웨이 개발
  - React + Three.js + Konva 기반 3D/2D 운영 UI 구현
